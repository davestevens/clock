Example 1

Create a default Clock.
Displays current time, (hours, minutes and seconds).
Defaults to appending to 'document.body', pass '{ el: HTML#element }' to append to specific element in the page.

Code:

new Clock();

Example 2

Customize time display.
Passing a custom function for 'calculate_example' allows you to override the timing.
This example uses Seconds and Milliseconds to calculate the hand position.

Code:

// Configure clock display
var minute_clock = function () {
  var date, time;
  date = new Date();
  time = date.getMilliseconds() + (date.getSeconds() * 1000);
  return (time * ((2 * Math.PI) / (60 * 1000)));
};

new Clock({
  calculate_angle: minute_clock
});

Example 3

Use custom styling/sizes

Code:

new Clock({
  style: {
    view: {
      size: 150,
      border: {
        size: 2
      }
    },
    hand: {
      size: 4,
      color: "rgba(255, 0, 255, 0.5)"
    },
    clock: {
      size: 400,
      image: "//oclockpress.com/files/clocktemplate.gif"
    }
  }
});
